# SHORTX | Enterprise URL Shortener Workspace

SHORTX is a production-ready, React Single-Page Application (SPA) designed to solve URL shortening and traffic distribution insights. The user interface has been built using a premium, minimalist design system modeled after Wix Studio and Wix Enterprise layouts.

## Quickstart

To run the development server locally:
```bash
# Navigate to the frontend directory
cd frontend

# Install package dependencies
npm install

# Start local server
npm run dev
```

To run build and lint validations:
```bash
# Lint checks
npm run lint

# Compile production-ready build bundle
npm run build
```
